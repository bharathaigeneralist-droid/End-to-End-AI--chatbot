2️⃣ Intelligent Chatbot System

This is the main highlight of your project 🔥

How it works:
Uses Sentence Transformers model (all-MiniLM-L6-v2)
Converts questions into vectors (embeddings)
Stores all Q&A in vector form
When user asks a question:
Converts input into vector
Compares with stored questions
Finds most similar question
Returns the best answer

👉 Uses cosine similarity for matching

3️⃣ Chatbot Features
⚡ Fast response (<10 ms)
